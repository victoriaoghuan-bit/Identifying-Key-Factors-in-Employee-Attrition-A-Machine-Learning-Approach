
# Identifying Key Factors in Employee Attrition: A Machine Learning Approach

## 📌 Overview

This project uses machine learning techniques to analyze employee data and identify the key factors contributing to employee attrition. The goal is to help organizations understand why employees leave and enable data-driven retention strategies.

## 🎯 Objectives

* Analyze employee attributes to understand attrition patterns
* Build predictive models to identify employees at risk of leaving
* Determine the most influential factors affecting attrition

## 📊 Dataset

The dataset includes employee information such as:

* Demographics (age, gender, education)
* Job-related factors (role, department, salary, years at company)
* Work environment (job satisfaction, overtime, work-life balance)

## 🛠️ Methodology

* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA)
* Feature engineering and selection
* Machine Learning models:

  * Logistic Regression
  * Decision Tree
  * Random Forest
  * XGBoost (optional)

## 📈 Results

* Model performance evaluated using accuracy, precision, recall, and F1-score
* Feature importance analysis highlights key attrition drivers such as:

  * Overtime
  * Job satisfaction
  * Monthly income
  * Work-life balance

## 📌 Key Insights

* Employees working overtime are more likely to leave
* Low job satisfaction strongly correlates with attrition
* Compensation and work-life balance are critical retention factors
